Portfolio website

Project Overview:
This project was to make a portfolio so that i would have a place where i can easily display previous projects.

Goal:
The goal of this project was to create a website where i can display my previous works, and also describe how i am as a person.

My Role & Contributions:
I did everything myself

Technologies Used:
HTML, CSS, Javascript.

Challenges & Solutions:
I initially had an issue with the header not being centered on the first page when compared to all of the other pages. I found out that this was due to the scroll bar. This was fixed however by ensuring that the vertical scrollbar is always visable.

I also faced a challenge of hosting the website. I ended up using github pages for this since its free and easy to set up. In the future i want to host it on a server, so that i can get my own domain.

Results:
The result of this is the page you are currently on!
