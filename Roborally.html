Robo Rally

Project Overview: Robo rally is a board game in which the players control different robots, that they then control by placing down cards that say "Go forward", "Go left" and so on. The goal of the game is to get to the end of the course before your compettition does.

Goal: We were tasked with making a Java version of Robo rally. This version behave similar to the board game version and include a way to play with a friend.

My Role & Contributions: Team management, Design of the board, Creating the board, Coding of player movement, Coding of some obstacles.

Technologies Used: Java, Draw.IO, RESTful API (Spring boot), MoSCoW model, JSON, Databases, JUnit tests.

Challenges & Solutions: We had a problem with getting player movement to work correctly, however this was solved by making sure that each tile on the board also had a heading. This made it possible to always be able to call and move the player according to the tile they are standing on.
