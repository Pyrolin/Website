Card Filtering app

Project Overview: The idea behind this project was to create an app that allows the user to search, filter and favorite cards from the card game "Magic the gathering".

Goal: The goal of this project was to make it easy for users to find the cards they want to play with or collect and to save their favorites for quick access later. Users can filter cards by various attributes to find exactly what they need. This functionality was implemented using an API.

My Role & Contributions: I served as the Project Leader, ensuring all team members stayed active and on schedule. Additionally, I was responsible for UI design, coding the user interface, and creating various app scenes.

Technologies Used: Android Studio, Figma, Kotlin, Scryfall API.

Challenges & Solutions: Initially, we had a problem with getting the filtering to work correctly. After examining exactly how the API pulls data, we learned that filtering parameters are added to the URL. For example, to search for blue cards we add something to the URL. This approach allowed us to only fetch the cards that matched the user's selected criteria.

Results: We ended up with a working prototype that performed really well, successfully filtering and displaying even the more obscure cards.
